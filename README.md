## JS Class Overview

This track covers core JavaScript language features followed by practical browser APIs.

### Language Basics (code files)
- Variables: [variables.js](./js-class/variables.js)
- Numbers & Operators: [numbers-operator.js](./js-class/numbers-operator.js)
- Functions & Scope (incl. closures, currying, callbacks): [function.js](./js-class/function.js)
- Arrays & Methods (map, filter, reduce, sort): [arrays.js](./js-class/arrays.js)

### Web APIs (browser demos)
- DOM API interactive demo: [dom.js](./js-class/web%20apis/dom.js) with UI in [index.html](./js-class/web%20apis/index.html) and styles in [styles.css](./js-class/web%20apis/styles.css)
- CSSOM API demo: [cssom.js](./js-class/web%20apis/cssom.js)
- Web Storage (localStorage/sessionStorage): [web-storage](./js-class/web%20apis/web-storage/)
- IndexedDB basics: [indexdb](./js-class/web%20apis/indexdb/)

### How to Try the Demos
1. Open [js-class/web apis/index.html](./js-class/web%20apis/index.html) in your browser.
2. Use the buttons on the page to run DOM/CSSOM examples.
3. Open DevTools Console to observe logs and results.
4. For the language basics, open each `.js` file and run snippets in the browser console or with Node.

### Suggested Learning Order
1. Start with variables → numbers/operators → functions → arrays.
2. Move to DOM → events → CSSOM → storage → IndexedDB.
3. Build a small app that persists data using Web Storage or IndexedDB.
