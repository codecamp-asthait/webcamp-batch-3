CSS Variables (Custom Properties)

Introduction

CSS Variables allow you to store and reuse values throughout your stylesheet. They're powerful, dynamic, and can be changed with JavaScript!

Basic Usage

Using --primary-color variable
Using --secondary-color variable
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
}

.example {
    background: var(--primary-color);
}
