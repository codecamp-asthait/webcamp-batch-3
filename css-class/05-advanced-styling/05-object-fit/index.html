Object-fit and Object-position

Introduction

The object-fit property controls how replaced elements (like images and videos) are resized to fit their container. object-position controls the positioning of the object within its container.

Object-fit Values

fill

fill - Stretches to fill container

contain

contain - Fits entire object

cover

cover - Fills container, may crop

none

none - Original size

scale-down

scale-down - Like none or contain
