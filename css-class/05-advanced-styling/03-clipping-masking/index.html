CSS Clipping and Masking

Introduction

CSS clipping and masking allow you to hide portions of elements, creating interesting visual effects. clip-path creates geometric shapes, while mask uses images or gradients to control visibility.

Basic clip-path Shapes

Circle

clip-path: circle(50%)

Ellipse

clip-path: ellipse(60% 40%)

Polygon

clip-path: polygon(...)

Inset

clip-path: inset(20%)
