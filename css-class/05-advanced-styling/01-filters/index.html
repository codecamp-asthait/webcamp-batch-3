CSS Filters and Backdrop Filters

Introduction

CSS filters allow you to apply visual effects like blur, brightness, contrast, and more to elements. Backdrop filters apply these effects to the area behind an element, creating beautiful glassmorphism effects.

Filter Functions

Blur

filter: blur(5px)

Brightness

filter: brightness(1.5)

Contrast

filter: contrast(150%)

Grayscale

filter: grayscale(100%)

Hue Rotate

filter: hue-rotate(90deg)

Invert

filter: invert(100%)

Opacity

filter: opacity(50%)

Saturate

filter: saturate(200%)

Sepia

filter: sepia(100%)

Combining Multiple Filters

Blur + Brightness

filter: blur(3px) brightness(1.2)

Grayscale + Contrast

filter: grayscale(50%) contrast(120%)

Multiple Effects

filter: blur(2px) brightness(1.1) saturate(150%)
