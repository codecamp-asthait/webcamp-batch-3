CSS Modules

What are CSS Modules?

CSS Modules is a CSS file in which all class names and animation names are scoped locally by default.

This solves the global namespace problem in CSS and enables safe, reusable component styling.

Problems CSS Modules Solve

Global Namespace

All CSS classes are global, causing potential naming conflicts

Cascade Issues

Unintended style cascading from other components

Refactoring Difficulty

Hard to know which styles are safe to remove

Dependencies

Unclear which CSS affects which elements

How CSS Modules Work

1. Write Local CSS

.button {
    padding: 10px 20px;
    background: blue;
}

2. Import in Component

import styles from './Button.module.css';

3. Use as Object

<button className={styles.button}>

4. Compiled Output

.Button_button__x2k9 {
    padding: 10px 20px;
    background: blue;
}

Example: Button Component

Button.module.css

.primary {
    background: #667eea;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}

.primary:hover {
    background: #764ba2;
}

.secondary {
    background: white;
    color: #333;
    border: 2px solid #667eea;
}

Button.jsx

import styles from './Button.module.css';

export function Button({ variant = 'primary' }) {
    return (
        <button className={
            styles[variant]
        }>
            Click Me
        </button>
    );
}

Key Features

Local Scope

All class names are locally scoped by default

Explicit Dependencies

Clear which styles belong to which component

Dead Code Removal

Easy to identify and remove unused styles

Composition

Import and compose styles from other modules

Framework Support

Create React App

Built-in support, use .module.css

Next.js

Built-in support for CSS Modules

Webpack

Configure css-loader with modules

Vite

Use .module.css out of the box

Best Practices

Use camelCase for class names in modules
Keep styles close to components
Use composition for shared styles
Avoid deep nesting in selectors
Document dependencies between modules
