Sass Introduction

What is Sass?

Sass (Syntactically Awesome Stylesheets) is a CSS preprocessor that extends CSS with powerful programming features like variables, nesting, mixins, and functions.

Sass code is compiled into standard CSS that browsers understand.

Key Features

Variables

Store and reuse values throughout your stylesheets.

$primary-color: #667eea;

Nesting

Organize related selectors in a hierarchical structure.

.card { &:hover { ... } }

Mixins

Create reusable blocks of CSS declarations.

@include flex-center;

Functions

Perform calculations and transformations on values.

scale($base-size * 2)

Variables Example

Scss

$primary: #667eea;
$spacing: 16px;

body {
    color: $primary;
    padding: $spacing;
}

Compiled CSS

body {
    color: #667eea;
    padding: 16px;
}

Getting Started

  1. Install Node.js from nodejs.org
  2. Install Sass: npm install -g sass
  3. Create a .scss file
  4. Compile: sass input.scss output.css
  5. Or use watch mode: sass --watch .
