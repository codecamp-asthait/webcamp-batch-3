Less - Leaner Style Sheets

What is Less?

Less (Leaner Style Sheets) is a dynamic preprocessor for CSS. It extends CSS with dynamic behavior such as variables, mixins, operations, and functions.

Less is similar to Sass but with a different approach and syntax.

Comparison: Less vs Sass

Less

  • CSS-like syntax
  • Client and server compilation
  • Faster to learn
  • Good for web projects
  • Similar to original CSS

Sass

  • More powerful features
  • Server-side only
  • Steeper learning curve
  • Better for complex projects
  • More like programming

Variables Example

Less Code

@brand-color: #667eea;
@spacing: 8px;

.button {
    background: @brand-color;
    padding: @spacing * 2;
}

Compiled CSS

.button {
    background: #667eea;
    padding: 16px;
}

Installation & Usage

Step 1: Install Node.js

Download from nodejs.org

Step 2: Install Less

npm install -g less

Step 3: Create .less Files

touch styles.less

Step 4: Compile

lessc styles.less styles.css

Step 5: Or Watch Changes

lessc --watch styles.less styles.css

Advantages of Less

Easy Syntax

Very similar to CSS, making it easy for developers to learn

Flexible Compilation

Can be compiled client-side or server-side

Powerful Features

Variables, mixins, nesting, functions all included

Community Support

Active community with great documentation
