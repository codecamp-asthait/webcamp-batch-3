# Basic CSS Properties

## Overview
This lesson covers essential CSS properties that you'll use frequently in web development.

## Common Text Properties

### Font Properties
- `font-family` - Specifies the font
- `font-size` - Sets the text size
- `font-weight` - Controls boldness (normal, bold, 100-900)
- `font-style` - normal, italic, oblique

### Text Properties
- `text-align` - Aligns text (left, right, center, justify)
- `text-decoration` - Adds decorations (underline, overline, line-through)
- `text-transform` - Controls capitalization (uppercase, lowercase, capitalize)
- `line-height` - Sets spacing between lines
- `letter-spacing` - Space between characters
- `word-spacing` - Space between words

## Spacing Properties

### Margin
- Controls space OUTSIDE an element
- `margin-top`, `margin-right`, `margin-bottom`, `margin-left`
- Shorthand: `margin: top right bottom left`

### Padding
- Controls space INSIDE an element
- `padding-top`, `padding-right`, `padding-bottom`, `padding-left`
- Shorthand: `padding: top right bottom left`

## Border Properties

```css
border: width style color;
```

- `border-width` - Thickness of border
- `border-style` - solid, dashed, dotted, double, none
- `border-color` - Color of border
- `border-radius` - Rounds corners

## Width and Height

- `width` - Sets element width
- `height` - Sets element height
- `max-width` - Maximum width
- `min-width` - Minimum width
- `max-height` - Maximum height
- `min-height` - Minimum height

## Display Property

- `display: block` - Takes full width available
- `display: inline` - Takes only necessary width
- `display: inline-block` - Hybrid of both
- `display: none` - Hides element completely

## Opacity and Visibility

- `opacity` - 0 (transparent) to 1 (opaque)
- `visibility` - visible, hidden (still takes space)

## Cursor

- `cursor: pointer` - Hand pointer
- `cursor: default` - Default arrow
- `cursor: text` - Text selection cursor
- `cursor: not-allowed` - Disabled indicator

## Practice Exercises

1. Create a button with padding, border, and border-radius
2. Style a paragraph with custom font, line-height, and letter-spacing
3. Create a card with margin, padding, and box-shadow
