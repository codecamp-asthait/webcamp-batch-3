Code Examples

Auto-Fit Grid

.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}

Flexible Container

.container {
    width: clamp(320px, 90%, 1200px);
    padding: clamp(1rem, 2vw, 3rem);
    margin: 0 auto;
}
