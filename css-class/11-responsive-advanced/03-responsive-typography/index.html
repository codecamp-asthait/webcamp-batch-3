Font Sizing Strategies

Breakpoint-Based Scaling

/* Mobile First */
h1 { font-size: 1.5rem; }
h2 { font-size: 1.3rem; }
p { font-size: 1rem; }

/* Tablet */
@media (min-width: 768px) {
    h1 { font-size: 2rem; }
    h2 { font-size: 1.6rem; }
}

/* Desktop */
@media (min-width: 1024px) {
    h1 { font-size: 2.5rem; }
    h2 { font-size: 2rem; }
}

Fluid Typography with calc()

Text that scales smoothly without breakpoints:

This text scales fluidly between breakpoints using CSS calc() for smooth transitions.

h1 {
    font-size: calc(1.5rem + (2.5 - 1.5) * (100vw - 320px) / (1280 - 320));
    max-width: 1280px;
}

Typography Best Practices

Use rem for Base

Set font sizes in rem for predictable scaling across the entire site.

Maintain Line Height

Keep line-height between 1.2-1.6 for optimal readability and spacing.

Test Readability

Verify text is readable on all screen sizes and in different lighting.

Hierarchy Matters

Use consistent font-weight and size ratios for visual hierarchy.
