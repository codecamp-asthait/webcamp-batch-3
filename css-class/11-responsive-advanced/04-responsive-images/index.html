Responsive Images

Serving Images for Every Device

Responsive images optimize file size and visual quality across different screen sizes and resolutions.

Basic Responsive Images

max-width: 100%; height: auto;

The basic approach: let images scale with their containers.

Picture Element

The picture element serves different images for different conditions:

Art Direction Example

<picture>
    <source media="(max-width: 479px)" 
            srcset="image-mobile.jpg">
    <source media="(min-width: 768px)" 
            srcset="image-desktop.jpg">
    <img src="image-default.jpg" alt="Image">
</picture>

Srcset and Sizes

Serve different resolutions based on device capabilities:

Resolution-Based Srcset

<img src="image.jpg"
     srcset="image-small.jpg 480w,
             image-medium.jpg 768w,
             image-large.jpg 1280w"
     sizes="(max-width: 600px) 100vw,
            50vw"
     alt="Responsive image">

480w

Mobile (up to 479px)

768w

Tablet (480-767px)

1280w

Desktop (768px+)

Progressive Format Support

     alt="Description"

Responsive Image Checklist

max-width: 100% on all images
height: auto or aspect-ratio set
Descriptive alt text for accessibility
srcset for multiple resolutions
picture element for art direction
Modern formats with fallbacks
Lazy loading for off-screen images
Testing on multiple devices
