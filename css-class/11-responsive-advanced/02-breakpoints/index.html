Understanding Breakpoints

What are Breakpoints?

Breakpoints are specific screen widths where your responsive design adapts. They're essential for creating layouts that work across different devices.

Common Breakpoints

XS (Extra Small)

0px - 479px

Phones

SM (Small)

480px - 767px

Large phones, tablets

MD (Medium)

768px - 1023px

Tablets

LG (Large)

1024px - 1279px

Small desktops

XL (Extra Large)

1280px+

Large desktops

Media Query Examples

Mobile First Approach

/* Mobile (XS) - Default */
.container { width: 100%; }

/* Tablet (MD) */
@media (min-width: 768px) {
    .container { width: 90%; }
}

/* Desktop (LG) */
@media (min-width: 1024px) {
    .container { max-width: 1000px; }
}

Best Practices

  • Start with mobile-first design
  • Use min-width for progressive enhancement
  • Keep breakpoints consistent
  • Test on real devices, not just browser
  • Keep 3-5 main breakpoints
  • Document your breakpoint strategy
  • Base breakpoints on content, not devices
