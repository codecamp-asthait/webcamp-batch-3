Animation Optimization Tips

✅ Use Transform and Opacity

These properties are GPU-accelerated and don't trigger layout or paint.

✅ Avoid Layout Properties

Don't animate width, height, margin, padding - they trigger layout recalculation.

✅ Use Will-Change Sparingly

Hint the browser about upcoming changes, but don't overuse.

✅ Limit Simultaneous Animations

Too many animations can impact performance.
