CSS Writing Modes

Horizontal Writing (Default)

This is the default horizontal-tb writing mode. Text flows from left to right and top to bottom.

writing-mode: horizontal-tb

Vertical Right-to-Left

This uses vertical-rl writing mode. Text flows from top to bottom, right to left.

writing-mode: vertical-rl

Vertical Left-to-Right

This uses vertical-lr writing mode. Text flows from top to bottom, left to right.

writing-mode: vertical-lr

Text Orientation: Mixed

This demonstrates text-orientation: mixed in vertical mode. Different character types rotate differently.

writing-mode: vertical-rl | text-orientation: mixed

Text Orientation: Upright

This shows text-orientation: upright. Characters remain upright even in vertical mode.

writing-mode: vertical-rl | text-orientation: upright

Combined Layout

Vertical

Content in vertical writing mode

Horizontal

Content in standard horizontal mode. You can mix writing modes in the same page for creative layouts.

Logical Properties Example

Using logical properties (inline-start, block-start) ensures correct spacing regardless of writing mode. These adapt to the current writing direction.
