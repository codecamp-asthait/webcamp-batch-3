## How to Use This Course

1. Start with Module 1 and work through each topic sequentially
2. Open the HTML file in each folder to see the examples
3. Modify the CSS files to experiment and learn
4. Each folder contains:
   - `index.html` - Example HTML structure
   - `styles.css` - CSS examples and exercises
   - `README.md` - Topic explanation and notes
