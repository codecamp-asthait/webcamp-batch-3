Floats and Clearing

Introduction

The float property was originally designed for wrapping text around images, but it became a popular layout technique before Flexbox and Grid. The clear property prevents elements from floating next to floated elements.

Note: While modern layouts use Flexbox and Grid, understanding floats is important for legacy code and specific use cases like text wrapping around images.

Basic Float Values

Float Left

This text wraps around the floated element. The float property removes the element from the normal document flow and positions it to the left or right of its container. Text and inline elements will wrap around it.

Float Right

This text wraps around the floated element on the right side. Notice how the text flows around the floated box. This is the original purpose of floats - to allow text to wrap around images or other content.
